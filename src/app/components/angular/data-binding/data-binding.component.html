PROPERTY BINDING

Cần truyền data cho một property nào đó!

Đối với DOM(Document Object Model) chúng ta có 2 khái niệm khác nhau là property và attribute.

Khi browser load trang web, nó sẽ parse phần HTML và xây dựng nên một cây DOM để biểu diễn phần HTML đang được dựng và cho phép tương tác với phần HTML đó bằng Javascript thông qua DOM.

Attribute được dùng để chỉ những gì được đặt vào phần opening tag còn lại là property của object. Các attribute sẽ được lưu vào property attribute của node tương ứng. Có những attribute sẽ được map sang thành property tương ứng, ví dụ như type và value(như trên), có những attribute không được map sang ví dụ: class sẽ thành className, chúng không có quan hệ 1-1 với nhau.
